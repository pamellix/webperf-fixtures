# Speed CLS Emulator

Сайт для моделирование различных CLS (Cumulative Layout Shift), с настройкой через query параметр.

## Query параметры

- `CLS`: целевое значение CLS (например, `?CLS=0.25` или `?CLS=0.8`)

## Как работает

После первой отрисовки страница программно вставляет сверху «баннер» высотой примерно `CLS * viewportHeight`, из‑за чего весь контент смещается вниз и формируется layout shift.
